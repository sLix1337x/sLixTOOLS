# sLixTOOLS - Free Online File Conversion Tools

A collection of free, easy-to-use online tools for all your file conversion needs. Built with modern web technologies for a seamless user experience.

## 🌟 Features

- **Video to GIF Converter** - Convert videos to high-quality GIFs with customizable settings
- **GIF Compressor** - Reduce GIF file size while maintaining quality
- **Modern UI/UX** - Clean, responsive design with smooth animations
- **Privacy Focused** - All processing happens in your browser
- **No Watermarks** - Get clean results without any branding

## 🛠️ Technologies Used

- ⚡ Vite - Next Generation Frontend Tooling
- 🟦 TypeScript - Type-Safe JavaScript
- ⚛️ React 18 - A JavaScript library for building user interfaces
- 🎨 shadcn/ui - Beautifully designed components
- 🎨 Tailwind CSS - A utility-first CSS framework
- 🎭 Framer Motion - Animation library for React
- 🌐 Three.js (R3F) - 3D graphics in the browser
- 🔄 Lenis - Smooth scroll library
- 📱 PWA Support - Offline capability with Vite PWA Plugin
- 📱 Fully Responsive - Works on all devices

## 🚀 Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:5173](http://localhost:5173) in your browser
